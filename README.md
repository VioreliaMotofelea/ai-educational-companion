# AI Educational Companion – Research Project

This repository contains the experimental code and documentation for my Research Project
and future BSc thesis: an AI-powered educational companion integrating gamification,
personalized recommendations, and intelligent scheduling.

## Structure
- data/ – synthetic CSV datasets
- src/ai_core – core logic (recommender, scheduler, gamification)
- experiments/ – scripts to run initial experiments
- tests/ – unit tests
- docs/ – research reports (Lab3, Lab5_8, etc.)

## How to run
pip install -r requirements.txt
python experiments/recommender_experiment.py
